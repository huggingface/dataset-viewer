# Splits

A dataset generally contains multiple *[splits](https://huggingface.co/docs/datasets/load_hub#splits)*, a specific subset of a dataset like `train` and `test`. The dataset can also contain *[configurations](https://huggingface.co/docs/datasets/load_hub#configurations)*, a sub-dataset of the larger dataset.

See the [documentation](https://huggingface.co/docs/datasets) of the [ðŸ¤— Datasets](https://github.com/huggingface/datasets) library to read more in depth about the concepts.

To get the list of splits and configurations of a dataset:

  ```
  https://datasets-server.huggingface.co/splits?dataset={dataset_name}
  ```

<Tip warning={true}>
 Currently, the API only returns rows of the <a href="https://huggingface.co/docs/datasets/stream">"streamable" datasets</a>. By loading a dataset in streaming mode, the first rows can be extracted without downloading the whole dataset.
</Tip>
