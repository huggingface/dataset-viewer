# API reference

The base URL of the REST API is

  ```
  https://datasets-server.huggingface.co
  ```

It provides the following endpoints:

| Endpoint | Description | Query parameters |
| --- | --- | --- |
| /splits GET | Get the list of configurations and splits of a dataset. | `dataset`: name of the dataset |
| /rows GET | Get the first rows of a dataset split. | - `dataset`: name of the dataset<br>- `config`: name of the config<br>- `split`: name of the split |
| /valid GET | Get the list of datasets hosted in the Hub and supported by the datasets server. | |
